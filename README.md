
## Cara Menjalankan Secara Lokal

1.  Clone repository ini ke komputer Anda.
2.  Buka folder proyek di editor kode Anda (seperti VS Code).
3.  Gunakan ekstensi **Live Server** (untuk VS Code) atau buka file `index.html` langsung di browser Anda.

## Deploy ke GitHub Pages

Website ini siap untuk di-deploy menggunakan GitHub Pages. Ikuti langkah-langkah yang ada di repository untuk mengaktifkannya.
